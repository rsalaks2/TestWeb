Image reflection

Šis ļauj bildi attēlot spoguļattēlā :

koda piemērs :

-webkit-box-reflect:right;
}

Transitions

Šis liek kaut kam mainīties, ja uz tā uzliek peles kursoru

koda piemērs :
#transimg{
transition: 2s;
}
#transimg:hover{
width: 500px;
height: 200px;
}

Informācijas avots : www.w3schools.com
